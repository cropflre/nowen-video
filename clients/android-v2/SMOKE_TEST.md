# Android V2 RC1 真机冒烟检查表

本文档用于 RC1 候选包的真机、模拟器和升级回归。自动化只覆盖首次启动的稳定基础链路；播放器、局域网、下载、系统生命周期和真实 NAS 环境必须按本检查表执行。

## 自动化基线

`Android V2 Device Smoke` workflow 在以下系统运行 `AppLaunchSmokeTest`：

- Android 8.0 / API 26
- Android 13 / API 33
- Android 15 / API 35

自动化验证：

1. Fresh install 可以启动到 V2 根页面；
2. 显示旧版并行迁移说明；
3. 迁移提示可以关闭；
4. 服务器设置页、扫码入口和手动添加入口可见。

本地执行：

```bash
./android/gradlew -p clients/android-v2 \
  connectedDebugAndroidTest \
  -Pandroid.testInstrumentationRunnerArguments.class=com.nowen.video.v2.AppLaunchSmokeTest
```

## P0：安装、启动与迁移

- [ ] 旧版 `com.nowen.video` 已安装时，V2 可以并行安装且出现两个独立图标。
- [ ] V2 首次启动显示迁移说明，不读取或修改旧版服务器与登录状态。
- [ ] 关闭迁移说明后进入服务器设置页。
- [ ] Debug 与 Release 包名符合预期；Debug 可与 V2 Release 并行安装。
- [ ] RC1 → 更高 RC 覆盖升级成功，服务器、会话和偏好保留。
- [ ] RC → Stable 覆盖升级成功，下载记录和本地文件保留。
- [ ] 更低 versionCode 安装被系统拒绝，不以卸载作为正式回滚方案。
- [ ] 卸载 V2 不影响旧版数据和运行。

## P0：服务器发现与认证

- [ ] 打开服务器页自动开始 mDNS 发现。
- [ ] mDNS 无结果时，私有网段常用端口探测能找到服务器。
- [ ] 局域网发现结果不会重复添加已保存服务器。
- [ ] 扫描 URL、Nowen JSON、`nowen-video://server` 三类二维码均可识别。
- [ ] 相机权限拒绝后有可恢复路径，不崩溃。
- [ ] 手动添加 HTTP、HTTPS、域名和 IPv4 地址成功。
- [ ] 无效地址、普通网页和不可达地址给出明确错误。
- [ ] 登录成功后进入首页。
- [ ] 首次强制改密后新密码与新会话立即生效。
- [ ] 退出登录后 Token 清除，但服务器资料保留。
- [ ] 多服务器切换时账号和 Token 不串用。

## P0：媒体浏览与导航

- [ ] 首页继续观看、媒体库、最近添加可以加载。
- [ ] 首页空数据和部分接口失败不会白屏。
- [ ] 媒体库首屏、追加页、空结果和失败重试正常。
- [ ] 类型、媒体库、关键词、标签、年份和排序组合生效。
- [ ] 快速连续修改筛选时旧分页请求被取消，没有重复或漏项。
- [ ] 电影、单集和剧集详情可以打开。
- [ ] 剧集季选择、单集列表、详情与播放跳转正确。
- [ ] 收藏、历史、合集和人物入口可以打开并返回。
- [ ] 系统返回键不会跳到错误页面或退出整个应用。

## P0：在线播放

至少准备以下测试媒体：Direct Play、Remux、HLS、内嵌多音轨、内嵌字幕、外挂 SRT/ASS、连续剧下一集。

- [ ] Direct Play 正常开始、暂停、拖动和退出。
- [ ] Remux 正常播放，服务端资源释放正确。
- [ ] HLS 正常播放，网络波动后能恢复或明确报错。
- [ ] 音轨、内嵌字幕和外挂字幕切换立即生效。
- [ ] 倍速和画面比例生效，并在重新进入播放器后保留。
- [ ] 播放进度从服务端恢复；达到 95% 后从头开始。
- [ ] 播放中每 10 秒上报，暂停、拖动、退后台和退出时即时补报。
- [ ] 播放结束显示下一集并执行 5 秒自动续播。
- [ ] 锁屏、解锁、耳机断开、切换前后台后播放器状态合理。
- [ ] 横竖屏切换不重复创建播放会话，不丢失当前位置。

## P0：离线下载与离线播放

- [ ] 默认仅 Wi-Fi 下载；切换允许移动网络后约束生效。
- [ ] 创建下载后前台通知、进度和存储占用正确。
- [ ] 暂停、继续、失败重试和删除正常。
- [ ] 杀进程后重新打开，排队和下载中任务可以恢复。
- [ ] 断网后任务保留；恢复网络后继续下载。
- [ ] HTTP Range 可断点续传；不支持 Range 时安全从头下载。
- [ ] 配额不足和设备空间不足时阻止任务并说明原因。
- [ ] HLS-only 媒体不会被误存为单文件。
- [ ] 完成文件在完全离线状态可以播放。
- [ ] 离线播放进度写入本地队列，恢复网络后补同步。
- [ ] 切换服务器或账号后看不到其他会话的下载任务。

## P0：网络与生命周期

- [ ] 播放过程中 Wi-Fi → 移动网络切换行为可预期。
- [ ] 在线页面断网后显示重试，不进入永久加载。
- [ ] 长时间后台后回到应用，会话和当前导航可恢复。
- [ ] 系统回收进程后从启动页恢复，不出现空白页。
- [ ] Token 过期时回到登录或刷新成功，不循环请求。
- [ ] 服务器不可达时不会删除已保存服务器或本地下载。

## P1：布局、无障碍与系统集成

- [ ] 手机单栏导航正常。
- [ ] 840dp 及以上设备显示媒体库双栏布局。
- [ ] 横屏、分屏和大字体下主要操作没有被裁切。
- [ ] 深色模式下文本、海报占位和对话框可读。
- [ ] 状态栏、导航栏和刘海安全区没有遮挡内容。
- [ ] TalkBack 可以读出主要按钮、导航和播放控制。
- [ ] 通知权限拒绝时下载失败提示明确。
- [ ] Android 13+ 通知、Android 14+ 前台服务限制下下载正常。

## RC 放行条件

- Android 8、13、15 自动启动冒烟全部通过；
- 三个 API 级别各至少一台真机或可信设备验证安装与启动；
- 所有 P0 用例为 `PASS`，或存在明确批准的非阻断已知问题；
- 没有崩溃、数据串号、凭据泄露、下载文件损坏或无法升级问题；
- APK/AAB 签名、versionCode、SHA-256 与发布记录一致；
- 回归结果附在 RC Issue 或 Release 草稿中。
